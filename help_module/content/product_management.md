# Product Management

This module allows you to manage the products available in your stores.

## Features
- **Manual Entry**: Add products one by one through the admin interface.
- **Modisoft Import**: Export your price book from Modisoft (make sure that the file contains minimul Scan code, item description and unit retail ) and import it here to update prices and names in bulk.
- **Automated Sync**: When a product's price, name, or supplier changes, any linked ESL tags will be automatically updated.

## Modisoft Import Process
1. Export your price book from Modisoft as an Excel file.(*Make sure that the file contains minimul Scan code, item description and unit retail*)
2. Navigate to the Products page in the SAIS platform.
3. Click on the "Import Modisoft" button.
4. Upload your file and preview the changes.
5. Confirm the import to apply changes to your store. (*Please note that when we import the file only the product description and item retail will be updated. There will be no changes done to the "is special" or "preferred supplier"*)
